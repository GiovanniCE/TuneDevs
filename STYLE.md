#Style guide

##Indentation
We will use 4 spaces for each indentation

##Maximum Columns
To improve readability, we will limit lines of code to 100 columns

##Horizontal alignment
NOT required, although permitted.

##Commenting
Provide javadoc style headers along with each method and class, and comment thoroughly throughout. 

##TODOs
Adding TODOs is permitted, but be sure to be specific in what needs to be done.
